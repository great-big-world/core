# core
Core mod for the Great Big World project.
 
# Subprojects
Great Big World is structured into subprojects for organizational purposes. These mods provide the majority of the content in the mod itself, utilizing the core mod as a central library.
- [Adventures](https://github.com/great-big-world/adventures)
- [Architect's Assembly](https://github.com/great-big-world/architects-assembly)
- [Flora & Fauna](https://github.com/great-big-world/flora-and-fauna)
- [Relics & Ruins](https://github.com/great-big-world/relics-and-ruins)
- [Swords & Shields](https://github.com/great-big-world/swords-and-shields)
- [The Alterworld](https://github.com/great-big-world/the-alterworld)

# Standalone Mods
Great Big World includes a variety of smaller mods packaged in the single jar file. These mods can be downloaded & used outside of the Great Big World project.
- [Lightning Flash](https://github.com/great-big-world/lightning-flash)
- [Rotatable Sculk](https://github.com/great-big-world/rotatable-sculk)
